共振

  • 判断贡献度和稳定性
    • Rule 1: 更多原子满足八电子结构
    • Rule 2: 更少电荷
    • Rule 3: 负电荷在电负性最大的原子上；正电荷在电负性最小的原子上
    • Rule 4: 具有更多共价键
    • Rule 5：π\piπ键出现在同周期元素中更稳定
    • Rule 6：芳香结构比非芳香更让稳定

酸碱理论

  • Arrhenius 酸碱理论：在水溶液中电离H+,OH−H^{+},OH^{-}H+,OH−
  • Bronsted-Lowry 酸碱理论: 给出质子（酸），接受质子（碱）
    • 解离常数：

      Ka=[A−][H+][HA]K_{a}=\frac{[A^{-}][H^{+}]}{[HA]} Ka​=[HA][A−][H+]​

      Kb=[HB+][OH−][B]K_{b}=\frac{[HB^{+}][OH^{-}]}{[B]} Kb​=[B][HB+][OH−]​

    • 局限：无法解释氧化物的酸碱性
  • Lewis 酸碱理论： 接受电子（酸），给出电子（碱）

立体化学

  • 构造异构和立体异构
    • 构象异构体：σ\sigmaσ键旋转引起的原子或基团的不同排列

烷烃

  • 结构: 构象，纽曼投影式
  • 物理性质：
    1. 物态
      - 1~4 Gas
      - 5~16 Liquid
      - above 16 Solid
    2. 相似相溶
    3. 熔点（M.P.）
      - M.P. increase with M
      - 变化趋势不同，（奇数/偶数），与是否能紧密堆叠有关
    4. 沸点（B.P.）
      - B.P. increase with M
      - Branch 支链多沸点低
  • 化学性质：
    1. 氧化反应Oxidation
    2. 烷烃热解（自由基反应）
    3. 卤化反应（自由基反应，hv or Δ\DeltaΔ）：活性顺序F2>Cl2>Br2>I2F_2>Cl_2>Br_2>I_2F2​>Cl2​>Br2​>I2​
      溴反应选择性高，伯<仲<叔，氯化反应比溴快
    4. 硝化反应（自由基，高温）
    5. 磺化反应（自由基，400°C）
  • 环烷烃：角张力，扭转张力
